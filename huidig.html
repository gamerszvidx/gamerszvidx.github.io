project 4: App

Voor project 4 hebben was de opdracht om een android applicatie te maken. Ook dit is volledig gedaan in java en wat SQL voor de database. Het uiteindelijke product was Bridge Ballot, een app om te kunnen zien of een van je favoriete bruggen open of dicht was. Hierop kan je je route aanpassen of mocht je zelf voor een dichte open brug staan dan kun je dit zelf ook aangeven voor andere gebruikers.

Project 3: Data

Project 3 was volledig gebaseerd op data, vrijwel alle data die wordt gebruikt in de gemaakte applicatie komt van twitter en wordt verwerkt tot informatie. Deze informatie wordt hierna gebruikt voor duidelijke weergaven in de vorm van bijvoorbeeld een Piechart, volledig in Java en Javafx
